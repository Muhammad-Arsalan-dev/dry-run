Stack Memory (LIFO)
📞
CALL label
Pushes return addr (SP−2), jumps to label.
↩
RET
Pops return addr into IP (SP+2). Resumes after CALL.
⬇
PUSH reg
SP = SP−2, memory[SP] = reg. Stack grows down.
⬆
POP reg
reg = memory[SP], SP = SP+2. Always reverse PUSH order.
📍
SP register
Always points to top of stack. Must return to base after proc.
🔁
LIFO Rule
Last pushed = first popped. PUSH A,B → POP B,A.
